# Actividades
  1. Obtener una archivo con un millon de datos.
  2. Aplicar una fórmula que permita gestionar los datos.
  3. Aplicar 2 tipos de ordenamiento en los datos.
  4. Cada tipo de ordenamiento debe imprimir el tiempo en el que se demora en ejecutarse.
  5. Mejor ordenamiento basado en el tiempo. Calcular cuanto se demora en procesar los dos tipos de ordenamiento y comparar. Cuál es el mejor ordenamiento?
  6. Realizar un multiprocesamiento(en python)
  7. Realizarlo en 3 procesadores diferentes para ver el tiempo de ejecución.
